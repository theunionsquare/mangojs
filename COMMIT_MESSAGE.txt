Major Features:
=============

1. OR Logic for Authorization Decorators (@OrAuth)
   - Added @OrAuth() decorator to enable OR logic between multiple auth checks
   - Supports flexible authorization: (Admin OR Owner) OR (Special Group)
   - Works with all auth decorators: @HasUserType, @HasGroups, @RequiresAccess, @RequiresOwnership
   - Default remains AND logic for backward compatibility

2. New Authorization Decorators
   - @RequiresOwnership: Validates resource ownership (prevents horizontal privilege escalation)
     * Supports single-value and array-based ownership checks
     * Custom validation logic support
     * Flexible parameter sources (params, query, body)
   - @OrAuth: Enables OR mode between multiple decorators
   - Enhanced @RequiresAccess with wildcard support for userTypes

3. Generic Cache Module (Reusable)
   - Extracted cache from auth to `/core/cache/` for application-wide use
   - Generic Cache<T> class with full type safety
   - TTL-based expiration with LRU eviction
   - Pattern-based cache invalidation
   - Performance metrics (hit rate, cache statistics)
   - Can be used for: database queries, API responses, computed results, etc.

4. Auth Configuration System (AuthConfig)
   - Global configuration for all auth decorators
   - Per-decorator configuration overrides
   - Configurable user context extraction
   - Custom error handlers
   - Conditional audit logging support
   - Cache configuration (TTL, max size)

5. Authorization Orchestrator
   - Centralized authorization logic with AND/OR mode support
   - Result caching with configurable TTL
   - Detailed error reporting with user context
   - Conditional audit logging based on configuration
   - Performance optimized with cache hit/miss tracking

Usage Examples:
==============

OR Logic:
```typescript
// Admin OR Owner can update
@OrAuth()
@HasUserType([AuthUserType.ADMIN])
@RequiresOwnership("partner")
async updatePartner(@Param("partnerId") partnerId: string) {}
```

Ownership Validation:
```typescript
// Prevent horizontal privilege escalation
@RequiresOwnership("partner")
async getPartnerData(@Param("partnerId") partnerId: string) {}

// Multi-tenant access
@RequiresOwnership("organization", {
  userField: "organizationIds",
  arrayField: true
})
async getOrgData(@Param("organizationId") id: string) {}
```

Generic Cache:
```typescript
import { Cache } from '@giusmento/mangojs-core';

const userCache = new Cache<UserData>({ ttl: 300000, maxSize: 10000 });
userCache.set('user:123', userData);
const cached = userCache.get('user:123');
```

Auth Configuration:
```typescript
AuthConfig.configure({
  userObjectPath: "session.user",
  enableAuditLog: true,
  cacheValidationResults: true,
  cacheTTL: 300000
});
```
