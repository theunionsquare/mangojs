    Class AuthContext

    Immutable authentication context

    Attached to requests after authentication middleware runs. Provides helper methods for authorization checks.

    Example

    // In middleware
    req.authContext = AuthContext.authenticated(user, 'jwt');

    // In controller
    if (req.authContext.hasUserType('ADMIN')) {
      // admin logic
    }

    if (req.authContext.hasPermission('users:delete')) {
      // delete user
    }
