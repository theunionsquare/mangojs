    Class AuthenticationError

    Authentication error (distinct from AuthorizationError)

    Authentication = "Who are you?" (401 Unauthorized) Authorization = "What can you do?" (403 Forbidden)

    Use this error when:

    • Token is missing, expired, or invalid
    • Credentials are incorrect
    • API key is invalid
    • Session has expired
